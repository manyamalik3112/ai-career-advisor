# 📊 AI Career Advisor Dashboard

## 🚀 Overview
This project is an interactive data analytics dashboard that analyzes Data Science job trends in India and provides personalized career recommendations based on user skills.

## 🎯 Key Features
- 📊 Salary analysis across job roles
- 📈 Experience vs salary trends
- 🤖 AI-based career recommendations
- 📚 Skill gap identification
- 🎯 Match confidence scoring

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## 📂 Dataset
Data Science Jobs in India (CSV dataset)

## ⚙️ How It Works
The system uses:
- Data cleaning and preprocessing
- GroupBy analysis for salary insights
- Weighted skill-matching algorithm
- Data merging to combine skills + salary insights
